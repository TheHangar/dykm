# Do You know Me
This is a learning project :
* create DB
* create back-end
Tech :
* PostgreSQL
* Nodejs

## Concept
Choose your favorite things (fruit, TV show, movies, Country, Crypto-currency ...), send to your friend a link and look up their score. 
**Do they know you ?**
